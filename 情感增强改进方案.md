## 😰 问题诊断

### 当前缺失的情感体验

1. ❌ **没有紧迫感** - 可以慢慢思考，没有时间压力
2. ❌ **失败代价低** - 错了扣几分而已，不痛不痒
3. ❌ **成功奖励弱** - 答对了只是"进入下一关"
4. ❌ **缺乏悬念** - 剧情平淡，没有转折
5. ❌ **反馈不够强烈** - 视觉/听觉反馈太温和
6. ❌ **没有竞争感** - 无法和他人比较

### 目标情感体验

- 😰 **紧张** - 时间压力和高风险决策
- 😤 **挣扎** - 面对困难选择的纠结
- 😱 **痛苦** - 失败时的挫败感
- 😄 **喜悦** - 成功时的多巴胺释放
- 🏆 **成就** - 完成挑战的自豪感
- 🔥 **兴奋** - 连续成功的状态流

## 方案A：通用情感增强方案

### 1️⃣ 添加倒计时压力系统 ⏰

**实现机制**：
```typescript
// 每个谜题都有倒计时
- 序章匹配：60秒内完成
- 第一章黑箱：90秒分析
- 第二章优选：120秒决策
- 第三章评估：150秒完成

// 时间奖励机制
- 剩余时间 > 50%：+10分 "神速"
- 剩余时间 > 30%：+5分 "迅捷"
- 剩余时间 < 10%：-5分 "险胜"
- 超时：-20分 + 强制进入"惩罚模式"
```

**UI设计**：
- 顶部中央：大号倒计时（红色警告 < 10秒）
- 倒计时低于30秒：屏幕边缘红色闪烁
- 倒计时归零：屏幕震动 + 警报音效

**情感效果**：
- 😰 **紧张** - 看着倒计时心跳加速
- 😅 **压力** - 必须快速做决策
- 😤 **挣扎** - 时间不够时的焦虑
- 🎉 **兴奋** - 最后几秒完成的刺激

### 1. 倒计时组件

```typescript
// src/components/Timer.tsx
import { useState, useEffect } from 'react';
import { motion } from 'motion/react';

interface TimerProps {
  duration: number; // 秒
  onTimeout: () => void;
  onComplete?: (remainingTime: number) => void;
}

export function Timer({ duration, onTimeout, onComplete }: TimerProps) {
  const [timeLeft, setTimeLeft] = useState(duration);
  const [isWarning, setIsWarning] = useState(false);

  useEffect(() => {
    if (timeLeft <= 0) {
      onTimeout();
      return;
    }

    if (timeLeft <= 30) {
      setIsWarning(true);
    }

    const timer = setInterval(() => {
      setTimeLeft(prev => prev - 1);
    }, 1000);

    return () => clearInterval(timer);
  }, [timeLeft, onTimeout]);

  const minutes = Math.floor(timeLeft / 60);
  const seconds = timeLeft % 60;
  const percentage = (timeLeft / duration) * 100;

  return (
    <motion.div
      className={`fixed top-4 left-1/2 -translate-x-1/2 z-50 ${
        isWarning ? 'animate-pulse' : ''
      }`}
      animate={isWarning ? { scale: [1, 1.1, 1] } : {}}
      transition={{ repeat: Infinity, duration: 1 }}
    >
      <div className={`px-6 py-3 rounded-lg border-2 ${
        isWarning 
          ? 'bg-red-900/80 border-red-500' 
          : 'bg-slate-900/80 border-slate-700'
      }`}>
        <div className="flex items-center gap-3">
          <span className="text-slate-400 text-sm">⏰ 剩余时间</span>
          <span className={`text-2xl font-bold font-mono ${
            isWarning ? 'text-red-400' : 'text-white'
          }`}>
            {String(minutes).padStart(2, '0')}:{String(seconds).padStart(2, '0')}
          </span>
        </div>
        <div className="mt-2 w-full h-1 bg-slate-800 rounded-full overflow-hidden">
          <motion.div
            className={`h-full ${
              isWarning ? 'bg-red-500' : 'bg-blue-500'
            }`}
            initial={{ width: '100%' }}
            animate={{ width: `${percentage}%` }}
          />
        </div>
      </div>
    </motion.div>
  );
}
```

### 4. 技术指标仪表盘

```typescript
// src/components/TechDashboard.tsx
import { motion } from 'motion/react';
import { TrendingUp, TrendingDown, Minus } from 'lucide-react';

interface Metric {
  label: string;
  value: number;
  unit: string;
  status: 'good' | 'warning' | 'danger';
  trend?: 'up' | 'down' | 'stable';
}

interface TechDashboardProps {
  metrics: Metric[];
  predictedRank: string;
}

export function TechDashboard({ metrics, predictedRank }: TechDashboardProps) {
  const getStatusColor = (status: string) => {
    switch (status) {
      case 'good': return 'text-green-400';
      case 'warning': return 'text-yellow-400';
      case 'danger': return 'text-red-400';
      default: return 'text-slate-400';
    }
  };

  const getTrendIcon = (trend?: string) => {
    switch (trend) {
      case 'up': return <TrendingUp className="w-4 h-4 text-green-400" />;
      case 'down': return <TrendingDown className="w-4 h-4 text-red-400" />;
      default: return <Minus className="w-4 h-4 text-slate-400" />;
    }
  };

  return (
    <motion.div
      initial={{ opacity: 0, x: 50 }}
      animate={{ opacity: 1, x: 0 }}
      className="fixed right-4 top-1/2 -translate-y-1/2 z-40"
    >
      <div className="bg-slate-900/90 border-2 border-slate-700 rounded-lg p-4 w-64">
        <h3 className="text-white font-bold mb-3 flex items-center gap-2">
          📊 项目仪表盘
        </h3>
        
        <div className="space-y-3">
          {metrics.map((metric, index) => (
            <motion.div
              key={index}
              initial={{ opacity: 0, x: 20 }}
              animate={{ opacity: 1, x: 0 }}
              transition={{ delay: index * 0.1 }}
              className="flex items-center justify-between"
            >
              <div className="flex-1">
                <div className="text-slate-400 text-xs">{metric.label}</div>
                <div className={`font-bold ${getStatusColor(metric.status)}`}>
                  {metric.value}{metric.unit}
                </div>
              </div>
              {metric.trend && getTrendIcon(metric.trend)}
            </motion.div>
          ))}
        </div>

        <div className="mt-4 pt-4 border-t border-slate-700">
          <div className="text-slate-400 text-xs mb-1">预测结局</div>
          <div className="text-white font-bold text-lg">{predictedRank}</div>
        </div>
      </div>
    </motion.div>
  );
}
```
