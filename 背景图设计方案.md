## 技术规格要求

- **分辨率**：1920x1080 或更高（16:9比例）
- **格式**：PNG（支持透明度，便于叠加效果）
- **文件大小**：建议500KB-1MB（优化后）
- **色彩模式**：RGB
- **主色调**：蓝色(#3b82f6)、灰色(#64748b)、深蓝(#1e3a8a)
- **辅助色**：暖白色(#fef3c7)用于灯光效果

## 生成工具推荐

1. **Midjourney**（推荐）
   - 使用 `--ar 16:9` 设置比例
   - 使用 `--style raw` 获得更写实的效果
   - 使用 `--v 6` 使用最新版本

2. **DALL-E 3**
   - 直接使用提供的提示词
   - 选择"landscape"横向格式

3. **Stable Diffusion**
   - 使用 Realistic Vision 或 DreamShaper 模型
   - 设置分辨率为 1920x1080
   - CFG Scale: 7-9

4. **Leonardo.ai**
   - 选择 PhotoReal 模式
   - 设置 16:9 比例

## 后期处理建议

生成图片后，建议进行以下处理：
1. 调整亮度，使其略暗（brightness 0.6-0.7）
2. 增加对比度（contrast 1.1-1.2）
3. 降低饱和度（saturation 0.8）
4. 添加轻微的蓝色色调滤镜
5. 确保图片整体偏暗，便于文字叠加显示

## 替代方案

如果AI生成效果不理想，也可以考虑：
1. 使用免费图库（Unsplash, Pexels）搜索相关工程/技术场景照片
2. 使用Photoshop/GIMP进行后期合成和调色
3. 委托专业设计师绘制
