## 技术设计实战：乡村照明系统改造项目

一款基于真实工程场景的互动式技术设计教学游戏。通过模拟真实的技术调查和设计流程，帮助学生掌握需求分析、系统设计、人机工程等核心知识点。

## 🎯 教学目标

- 掌握需求分析方法（用户需求、系统约束、问题定义）
- 理解黑箱法和筛选法的应用
- 学习技术语言规范化和标准化设计
- 掌握人机关系四要素（安全、高效、健康、舒适）

## 🎮 游戏特色

- ⏰ 时间压力系统 - 模拟真实项目deadline
- ⭐ 质量评级系统 - 5星工程质量评估
- 💬 专家评审反馈 - 权威专家实时指导
- 📊 技术指标仪表盘 - 数据驱动的决策支持

## 💻 技术栈

- React 19
- TypeScript
- Vite 6
- TailwindCSS 4
- Framer Motion

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 启动局域网服务器（多设备访问）
npm run dev -- --host
```

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
